Linux下PWN环境配置

发表于2023-03-19|更新于2023-04-15|Linux
|字数总计:1.2k|阅读时长:5分钟|阅读量:

更换镜像源

Linux 自带的源来自国外，有时候下载速度感人，十分难受，建议换成清华镜像

Ubuntu 22.04

注意看好版本，这是 22.04 版的 Ubuntu，其他版本在 sources.list 下加入的内容可能会不一样

cd /etc/apt # 进入apt目录下
vim sources.list # 编辑sources.list文件

# 加入下面两个----之间的内容（就当你会用vim了）
-----------------------------------------------------------------
# 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-security main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-security main restricted universe multiverse

# 预发布软件源，不建议启用
# deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-proposed main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-proposed main restricted universe multiverse
-----------------------------------------------------------------

 # 刷新三连
sudo apt-get update
sudo apt-get upgrade
sudo apt-get clean

Kali 2022.4

注意看好版本，这是 2022.4 版的 Kali，其他版本在 sources.list 下加入的内容可能会不一样

cd /etc/apt # 进入apt目录下
vim sources.list # 编辑sources.list文件

# 加入下面两个----之间的内容（就当你会用vim了）
-----------------------------------------------------------------
deb http://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free 
deb-src https://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free
-----------------------------------------------------------------

# 刷新三联
sudo apt-get update
sudo apt-get upgrade
sudo apt-get clean

Python 配置

安装 pip

wget https://bootstrap.pypa.io/get-pip.py  --no-check-certificate
python get-pip.py

修改默认 python 版本

sudo rm /usr/bin/python
# 例如我修改为python3.10
sudo ln -s /usr/bin/python3.10 /usr/bin/python

更换 pip 源

更换完下载源，该更换 pip 源了，在 Windows 下用 pip 安装过 python 库的都懂，这玩意不换没法用啊

  • 在命令行直接更改，打开终端：
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

CTF 配置

好了回归正题，可以开始安装 CTF 所需要的环境了

安装 32位 库

不安装别想运行 32位 的 ELF 可执行文件

sudo apt install libc6-dev-i386
sudo apt-get install lib32z1

安装 Capstone

一个轻量级的多平台架构支持的反汇编架构

git clone https://github.com/aquynh/capstone
cd capstone
make
sudo make install

安装 pwntools

通过 pip 安装或通过 git 安装，两种方法都可以，选其一即可

  • 通过 python 的 pip 安装
sudo apt-get install python3-pip
pip3 install --upgrade pip
sudo python3 -m pip install pwntools
  • 通过 git 安装
git clone https://github.com/Gallopsled/pwntools
cd pwntools
sudo python setup.py install
  • 验证安装
python # 打开python

# 输入下面的代码，Ⅰ或Ⅱ选其一即可
-------------------------------------------
# Ⅰ
import pwn
pwn.asm("xor eax,eax")
-------------------------------------------
# Ⅱ
from pwn import *
asm("xor eax,eax")
-------------------------------------------
# 如果输出的是这个，那就是安装成功了
输出：'1\xc0'
-------------------------------------------

安装 LibcSearcher

在做栈溢出相关的题时，经常会遇到构造 ROP 链的情况。若我们在 IDA 反编译之后没有看到 system 函数，栈题大概率需要泄漏 libc 库中的函数。此时，LibcSearcher 就是最好的选择，可以用它找到偏移地址

git clone https://github.com/lieanu/LibcSearcher.git
cd LibcSearcher
sudo python setup.py develop

安装 ROPgadget

用来找 gadget 的，这个安装 pwntools 时自带，如果没有的话可以再手动安装

git clone https://github.com/JonathanSalwan/ROPgadget.git
cd ROPgadget
sudo python setup.py install

安装 one_gadget

one_gadget 可以用来动态查找执行 libc 动态库中的 system 执行函数，可以一步到位。在做相关堆题时尤为有效
用来寻找 libc 库中的 execve('/bin/sh', NULL, NULL) 可以一个 gadget 就可以 getshell，需要注意 one_gadget 相关限制条件

sudo apt install ruby
sudo gem install one_gadget

安装 glibc-all-in-one

在做堆题时，经常遇到不同版本的 libc，这时 glibc-all-in-one 可以很好的派上用场，动态更改 elf 文件 libc 版本

git clone https://github.com/matrix1001/glibc-all-in-one.git
cd glibc-all-in-one
./update_list
cat list
./download <需要的 libc 版本>（例如：2.31-0ubuntu9_amd64）

安装 qemu

这是 arm 的 pwn 环境，前期可以不安装，但是终究是逃不过的，建议一步到位

sudo apt-get install qemu
sudo apt-get install qemu-system qemu-user-static binfmt-support

## 安装依赖库
sudo apt-get install -y gcc-arm-linux-gnueabi
sudo apt-get install qemu libncurses5-dev gcc-arm-linux-gnueabi build-essential gdb-arm-none-eabi synaptic gcc-aarch64-linux-gnu eclipse-cdt git

安装 Zsteg

做 MISC 需要，图像隐写神器

sudo gem install zsteg

安装 Basecrack

做 REVERSE、MISC 需要，Base 系列编码分析工具

git clone https://github.com/mufeedvh/basecrack.git
cd basecrack
pip install -r requirements.txt
python basecrack.py
