CTF配置

发表于2023-03-19|更新于2023-03-19|Linux
|字数总计:546|阅读时长:2分钟

安装 32位 库

sudo apt install libc6-dev-i386
sudo apt-get install lib32z1

安装 Capstone

一个轻量级的多平台架构支持的反汇编架构

git clone https://github.com/aquynh/capstone
cd capstone
make
sudo make install

安装 pwntools

  • 通过 python 的 pip 安装
sudo apt-get install python3-pip
pip3 install --upgrade pip
sudo python3 -m pip install pwntools
  • 通过 git 安装
git clone https://github.com/Gallopsled/pwntools
cd pwntools
sudo python setup.py install
  • 验证安装
python

-------------------------------------------
import pwn
pwn.asm("xor eax,eax")
-------------------------------------------
from pwn import *
asm("xor eax,eax")
-------------------------------------------
输出：'1\xc0'
-------------------------------------------

安装 LibcSearcher

在做栈溢出相关的题时，经常会遇到构造 ROP 链的情况。若我们在 IDA 反编译之后没有看到 system 函数，栈题大概率需要泄漏 libc 库中的函数。此时，LibcSearcher 就是最好的选择，可以用它找到偏移地址

git clone https://github.com/lieanu/LibcSearcher.git
cd LibcSearcher
sudo python setup.py develop

安装 ROPgadget

用来找 gadget 的，这个安装 pwntools 时自带，如果没有的话可以再手动安装

git clone https://github.com/JonathanSalwan/ROPgadget.git
cd ROPgadget
sudo python setup.py install

安装 one_gadget

one_gadget 可以用来动态查找执行 libc 动态库中的 system 执行函数，可以一步到位。在做相关堆题时尤为有效
用来寻找 libc 库中的 execve('/bin/sh', NULL, NULL) 可以一个 gadget 就可以 getshell，需要注意 one_gadget 相关限制条件

sudo apt install ruby
sudo gem install one_gadget

安装 glibc-all-in-one

在做堆题时，经常遇到不同版本的 libc，这时 glibc-all-in-one 可以很好的派上用场，动态更改 elf 文件 libc 版本

git clone https://github.com/matrix1001/glibc-all-in-one.git
cd glibc-all-in-one
./update_list
cat list
./download <需要的 libc 版本>（例如：2.31-0ubuntu9_amd64）

安装 qemu

这是 arm 的 pwn 环境，前期可以不安装，但是终究是逃不过的，建议一步到位

sudo apt-get install qemu
sudo apt-get install qemu-system qemu-user-static binfmt-support

# 安装依赖库
sudo apt-get install -y gcc-arm-linux-gnueabi
sudo apt-get install qemu libncurses5-dev gcc-arm-linux-gnueabi build-essential gdb-arm-none-eabi synaptic gcc-aarch64-linux-gnu eclipse-cdt git

安装 zsteg

图像隐写神器

sudo gem install zsteg

安装 basecrack

Base 编码分析工具

git clone https://github.com/mufeedvh/basecrack.git
cd basecrack
pip install -r requirements.txt
python basecrack.py
