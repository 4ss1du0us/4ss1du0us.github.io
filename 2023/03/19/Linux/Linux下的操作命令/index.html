Linux下的操作命令

发表于2023-03-19|更新于2023-03-29|Linux
|字数总计:483|阅读时长:1分钟|阅读量:

路径

一个点（./），代表本级目录
两个点（../），代表上级目录

查看文件内容

cat <文件名>

转到盘符/目录

cd E:
cd <路径名>

显示目录中的文件

ls <目录名>（仅显示可见文件）
ls -a（显示所有文件，包括隐藏文件）
ll（显示详细信息）

文件重命名

mv <文件名1> <文件名2>

1.文件被命名为 . 开头时，如 .flag，则 flag 为隐藏文件，ls 无法查看，需用 ls -a 才可以看到
2.将隐藏文件重命名去掉开头的 . 时，文件恢复可见

编辑文档

vim <文件名>
i（进入编辑模式）
esc（退出编辑模式）
:q（退出，但不保存）
:wq（保存并退出）
esc u（撤销之前的操作）

给文件增加可执行权限

chmod a+x <文件名>

运行 elf 可执行文件

./ <文件名>

运行 python 程序

python3 <文件名>

运行 C 代码

gcc <.c文件名> -o <编译生成的文件名>
./<编译生成的文件名>

复制 文件/文件夹

cp <源文件/源文件夹 路径> <复制到 文件/文件夹>

移动、重命名 文件/文件夹

mv <源文件/源文件夹 路径> <移动后的 文件/文件夹 路径>
mv <源文件名> <修改后的文件名>

卸载软件

1.--purge 是可选项，这个属性是将软件及其配置文件一并删除
sudo apt-get --purge remove <包名>
2.如果不需要删除配置文件，则不加 --purge 选项
sudo apt-get remove <包名>

删除文件/文件夹

rm -d <目录名>（删除一个空目录）
rm -r <目录名>（删除一个非空目录）
rm 文件名（删除文件）

安装 .deb 文件包

sudo dpkg -i <.deb 文件名>
