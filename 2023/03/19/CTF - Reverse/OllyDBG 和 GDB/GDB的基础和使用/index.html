GDB的基础和使用

发表于2023-03-19|更新于2023-04-14|OllyDBG&GDB
|字数总计:264|阅读时长:1分钟|阅读量:

GDB的安装与环境配置

安装 GDB

sudo apt-get install gdb

安装 gdb-peda

git clone https://github.com/longld/peda.git ~/peda
echo "source ~/peda/peda.py" >> ~/.gdbinit

下载 Pwngdb

git clone https://github.com/scwuaptx/Pwngdb.git

下载安装 pwndbg

git clone https://github.com/pwndbg/pwndbg 
cd ~/pwndbg
./setup.sh

配置 Pwngdb

cd ~
cp ~/Pwngdb/.gdbinit ~/
vim ~/.gdbinit
# 在 .gdbinit 的第二行插入：source ~/pwndbg/gdbinit.py
  • 运行 gdb，查看是否出错，是否显示 gdb-peda$
  • 若命令行在 gdb-peda$ 之前显示有错误，将 .gdbinit 第一行 source ~/peda/peda.py 注掉

GDB的操作指令

操作指令
调试程序gdb <文件名>
下断点b <函数名>
删除断点d
执行程序r
单步执行n
直接执行到程序结束c
查看寄存器的值i r <寄存器名>
修改寄存器的值set $<寄存器名> = <修改的值>
退出调试q
单步执行，遇到函数会进入s
将当前程序执行流跳转到指定行或地址j
打印变量或寄存器值p
运行到指定行停下来u
